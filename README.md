# HTML & JS Prework

A collection of exercises completed as prework for a software development bootcamp. Includes basic HTML pages and an introductory Express.js server.

## Learning Goals

This project is part of my journey to become a **Software Tester**. I use it as a reference to:

- Understand the basics of how web pages are structured (HTML)
- Learn how a simple backend server works with **Express.js**
- Practice reading and understanding code before testing it
- Identify what can be manually tested in a simple web app: page loads, links, content

## Contents

| File/Folder | Description |
|---|---|
| `helloworld.html` | First HTML page — basic structure and content |
| `profile.html` | A personal profile HTML page |
| `express/` | Intro to Node.js backend with Express |

## Running the Express Server

```bash
cd express
node index.js
```
